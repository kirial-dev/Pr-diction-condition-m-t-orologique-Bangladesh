La base de donnée chattogram Bangladesh regroupe  Six variables date, température, pluviométrie, l’humidité, la vitesse du vent et la condition climatique. L’objectif est de prédire la condition climatique (Sunny, Cloudy, Rainy, Heavy Rain) en fonction des autres features. Une variable season a été crée permettant de définir la saison au cours de l’année suivant la période de l’année. Aussi, la variable SPI , indice de précipitation standardisé a été créée, c’est un indice climatique couramment utilisé dans les travaux de recherches et dans le domaine agricole. 

Nous avons utilisé un modèle Random Forest Classifier pour prédire la condition climatique. Avec une précision de précision de 100%, nous sommes convaincu que c’est le meilleur modèle comparativement au XGboosting. Ensuite, nous sommes utilisé la bibliothèque Shap pour la visualisation, ce qui a permet de démonter que la variable pluviométrie et SPI ont contribué énormément dans notre modèle.
